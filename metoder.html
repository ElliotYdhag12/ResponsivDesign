1. Flexbox

Flexbox är en CSS-layoutmodell som gör det enkelt att placera och justera element på olika skärmstorlekar. Flexbox fungerar genom att definiera ett flex-kontainer och positionera innehållet inom den med flex-egenskaper.

Flexbox gör att elementen kan justera sig själva beroende på tillgängligt utrymme på skärmen.

2. CSS Grid

CSS Grid är en annan layoutteknik som gör det möjligt att skapa rutnätsbaserade layouter. Du kan definiera både rader och kolumner, vilket gör det perfekt för komplexa layoutstrukturer.

Grid gör det möjligt att placera element på bestämda platser i ett rutnät och anpassar sig bra till skärmstorlekar.

3. Media Queries

Media Queries är en teknik som gör det möjligt att använda olika CSS-regler beroende på enhetens skärmstorlek, upplösning eller andra egenskaper.

Med media queries kan du skapa specifika regler för olika enheter, vilket gör att din hemsida kan anpassa sig till olika skärmstorlekar (t.ex. mobil, tablet, desktop).
